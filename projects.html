Menu System

The python script creates a menu system which enables the user to choose from a list of options.

The options include:

  1. Find the volume of a box
  2. Count the amount of digits in a number
  3. Reverse a string of numbers
  4. Find the sum of numbers entered
  5. Find the Babylonian Square Root of a number
  6. Quit the menu

The user can run the menu item repeatedly until quitting and returning to the main menu to select another option or quit the program.

Guessing Game

The python script generates a random number from 1 to 50.

The user must enter a number repeatedly until the random number is guessed.

The program prints messages to let the user know if the guess was in the right range and continues to do so as the user gets closer and closer to the random number.
